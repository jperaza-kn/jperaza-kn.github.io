Idea

La ecuación de Bernoulli \[ y' + A(x)\,y = B(x)\,y^n \] se linealiza con el cambio \(v=y^{1-n}\) (para \(n\neq 1\)). En efecto: \[ v' + (1-n)A(x)\,v = (1-n)B(x), \] que es una ecuación lineal de primer orden.

Cómo se resuelve

Si \(n\neq 1\), usamos el cambio \(v=y^{1-n}\). Entonces: \[ v' + (1-n)A(x)\,v = (1-n)B(x), \] que es lineal. Se resuelve con factor integrante y luego se recupera \(y=v^{1/(1-n)}\).

Uso

Escribe \(A(x)\) y \(B(x)\) (sintaxis tipo LaTeX/Python), define \(n\), y pulsa Apply. Haz click en el plano para dibujar soluciones numéricas.

En el plano \((x,y)\), las isoclinas son las curvas donde \(y'=m\), es decir, \(B(x)y^n - A(x)y = m\). El applet muestra el campo de pendientes y la isoclina naranja para el valor \(m\).

Idea

The Bernoulli equation \[ y' + A(x)\,y = B(x)\,y^n \] becomes linear under the substitution \(v=y^{1-n}\) (for \(n\neq 1\)). One gets: \[ v' + (1-n)A(x)\,v = (1-n)B(x), \] which is a first-order linear ODE.

How to solve

If \(n\neq 1\), we use the substitution \(v=y^{1-n}\). Then: \[ v' + (1-n)A(x)\,v = (1-n)B(x), \] which is linear. We solve it with an integrating factor and then recover \(y=v^{1/(1-n)}\).

In the \((x,y)\) plane, isoclines are curves where \(y'=m\), i.e. \(B(x)y^n - A(x)y = m\). The applet shows the slope field and the orange isocline for the chosen \(m\).
