Descripción


            

              Resolvemos la ecuación de onda 1D con extremos fijos:
            


            

              \[
              \partial_{tt}u = k\,\partial_{xx}u,\qquad u(t,0)=u(t,10)=0.
              \]
            


            

              La condición inicial es un pulso gaussiano. Para que se mueva
              aproximadamente hacia la derecha con velocidad \(c=\sqrt{k}\), tomamos
              \(u(x,0)=u_0(x)\) y \(u_t(x,0)=-c\,u_0'(x)\).
              Las fronteras fijas producen reflexiones.
            


            
Expansión en senos


            

              La solución se escribe como
              \[
                u(x,t)=\sum_{n=1}^\infty \Big(a_n\cos(\omega_n t)+\frac{b_n}{\omega_n}\sin(\omega_n t)\Big)\,
                \sin\Big(\frac{n\pi x}{10}\Big),
              \]
              con \(\omega_n=c\,n\pi/10\).
              Puedes activar los primeros 10 armónicos para ver las contribuciones individuales.
            

          


          
          

            
Description


            

              We solve the 1D wave equation with fixed ends:
            


            

              \[
              \partial_{tt}u = k\,\partial_{xx}u,\qquad u(t,0)=u(t,10)=0.
              \]
            


            

              The initial condition is a Gaussian pulse. To make it move (approximately)
              to the right with speed \(c=\sqrt{k}\), we set
              \(u(x,0)=u_0(x)\) and \(u_t(x,0)=-c\,u_0'(x)\).
              Fixed boundaries enforce reflections.
            


            
Sine-series expansion


            

              The solution is
              \[
                u(x,t)=\sum_{n=1}^\infty \Big(a_n\cos(\omega_n t)+\frac{b_n}{\omega_n}\sin(\omega_n t)\Big)\,
                \sin\Big(\frac{n\pi x}{10}\Big),
              \]
              where \(\omega_n=c\,n\pi/10\).
              You can show the first 10 harmonics to visualize individual contributions.
