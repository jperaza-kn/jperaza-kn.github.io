Flujo del sistema

            

              Este applet visualiza el campo de flujo tridimensional del sistema lineal
              \[
                \dot{X} = AX,\qquad A\in\mathbb{R}^{3\times 3}.
              \]
            


            


            
Descripción

            
    
              
  • Cada punto entero del cubo \([-2,2]^3\) es una condición inicial.
    • 
              
  • Las curvas muestran la evolución entre \(t=-T\) y \(t=T\) (o solo \(t\in[0,T]\)).
    • 
              
  • Ajusta los deslizadores para cambiar la matriz \(A\).
    • 
              
  • Activa “autovectores/autovalores” para ver direcciones propias desde el origen.
    • 
            

          


          

            
System flow

            

              This applet visualizes the 3D flow field of the linear system
              \[
                \dot{X} = AX,\qquad A\in\mathbb{R}^{3\times 3}.
              \]
            


            


            
Description

            
    
              
  • Each integer point in \([-2,2]^3\) is used as an initial condition.
    • 
              
  • Trajectories show evolution on \(t\in[-T,T]\) (or only \(t\in[0,T]\)).
    • 
              
  • Use the sliders to modify the matrix \(A\) and the integration time \(T\).
    • 
              
  • Enable eigenvectors/eigenvalues to display principal directions from the origin.
    •
