Riccati equation — isoclines

\(y' + A(x)\,y + B(x)\,y^2 = C(x)\)

m = 0.00
-2 0 2
\(\text{isocline } y'=m\) 

        


        
        


          

            
          


          

            
Idea

            

              La ecuación de Riccati
              \[
                y' + A(x)\,y + B(x)\,y^2 = C(x)
              \]
              es no lineal. Si conoces una solución particular \(y_p(x)\), entonces el cambio
              \[
                y(x)=y_p(x)+\frac{1}{u(x)}
              \]
              reduce Riccati a una ecuación lineal de primer orden para \(u(x)\).
            

            

              En la figura, el campo direccional corresponde a
              \[
                y' = C(x)-A(x)\,y-B(x)\,y^2,
              \]
              y la isoclina naranja es el conjunto donde \(y'=m\).
            

          


          

            
Idea

            

              The Riccati equation
              \[
                y' + A(x)\,y + B(x)\,y^2 = C(x)
              \]
              is nonlinear. If you know one particular solution \(y_p(x)\), then the substitution
              \[
                y(x)=y_p(x)+\frac{1}{u(x)}
              \]
              turns Riccati into a first-order linear ODE for \(u(x)\).
            

            

              In the plot, the direction field is for
              \[
                y' = C(x)-A(x)\,y-B(x)\,y^2,
              \]
              and the orange curve is the isocline where \(y'=m\).
