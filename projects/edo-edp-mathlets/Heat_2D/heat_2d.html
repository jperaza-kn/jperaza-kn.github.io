Descripción

            

              Resolvemos la ecuación de calor en 2D con condiciones de borde de Dirichlet fijas:
            

            

              \[
                u_t = \Delta u,\qquad (x,y)\in[0,1]^2,\qquad u|_{\partial\Omega}=g.
              \]
            

            

              Los bordes se parametrizan con \(a,b,c,d\):
              \(\ u(0,y)=y^2+ay+b\), \(\ u(1,y)=y^2+cy+d\),
              y tomamos \(u(x,0)\) y \(u(x,1)\) lineales para que las esquinas coincidan.
            

            

              Al presionar Start, avanzamos hasta \(t=2\) (más lento) y verás cómo el calor
              suaviza el perfil interior hacia el estado estacionario armónico (la solución de Laplace con el mismo borde).
              La condición inicial \(u(x,y,0)\) se construye para respetar exactamente el borde y contener una perturbación interior
              que desaparece en \(\partial\Omega\).
            

          


          

            
Description

            

              We solve the 2D heat equation with fixed Dirichlet boundary data:
            

            

              \[
                u_t = \Delta u,\qquad (x,y)\in[0,1]^2,\qquad u|_{\partial\Omega}=g.
              \]
            

            

              The boundary is parameterized by \(a,b,c,d\):
              \(\ u(0,y)=y^2+ay+b\), \(\ u(1,y)=y^2+cy+d\),
              and we choose \(u(x,0)\), \(u(x,1)\) to be linear so corners match exactly.
            

            

              Press Start to run the evolution up to \(t=2\) (slower). The heat flow smooths the interior
              and relaxes toward the harmonic steady state (the Laplace solution with the same boundary).
              The initial condition \(u(x,y,0)\) is built to satisfy the boundary exactly while adding a purely interior bump
              that vanishes on \(\partial\Omega\).
