Descripción


            

              El sistema de Lotka–Volterra modela una interacción presa–depredador:
            


            

              \[
              \dot x = \alpha x - \beta x y,\qquad
              \dot y = \delta x y - \gamma y.
              \]
            


            
    
              
  • \(x(t)\): población de presas.
    • 
              
  • \(y(t)\): población de depredadores.
    • 
              
  • \(\alpha\): crecimiento natural de las presas.
    • 
              
  • \(\beta\): tasa de encuentro/predación.
    • 
              
  • \(\gamma\): mortalidad natural de depredadores.
    • 
              
  • \(\delta\): eficiencia de conversión (presas → depredadores).
    • 
            


            

              Arriba ves la trayectoria en el plano \((x,y)\) (retrato de fase).
              Abajo, las series temporales \(x(t)\) y \(y(t)\).
            

          


          
          

            
Description


            

              The Lotka–Volterra system models a predator–prey interaction:
            


            

              \[
              \dot x = \alpha x - \beta x y,\qquad
              \dot y = \delta x y - \gamma y.
              \]
            


            
    
              
  • \(x(t)\): prey population.
    • 
              
  • \(y(t)\): predator population.
    • 
              
  • \(\alpha\): prey intrinsic growth.
    • 
              
  • \(\beta\): predation rate.
    • 
              
  • \(\gamma\): predator intrinsic death.
    • 
              
  • \(\delta\): conversion efficiency (prey → predators).
    • 
            


            

              The top plot is the phase portrait \((x,y)\).
              The bottom plot shows the time series \(x(t)\) and \(y(t)\).
