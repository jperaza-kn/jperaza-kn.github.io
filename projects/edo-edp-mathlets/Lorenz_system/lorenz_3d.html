Description

            

              The Lorenz system is a classic model of convection that can show chaotic dynamics for certain parameters:
            


            

              \[
              \dot x=\sigma(y-x),\qquad
              \dot y=x(\rho-z)-y,\qquad
              \dot z=xy-\beta z.
              \]
            


            

              The parameters correspond (in the Rayleigh–Bénard setting) to:
            

            
    
              
  • \(\sigma\): the Prandtl number, ratio between momentum and thermal diffusivities.
    • 
              
  • \(\rho\): the Rayleigh number, the free convection parameter.
    • 
              
  • \(\beta\): related to the geometry / physical dimensions of the fluid layer (and dissipation).
    •
