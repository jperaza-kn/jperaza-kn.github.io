Idea

            

              Para un sistema autónomo \(\dot x=f(x,y),\ \dot y=g(x,y)\), una función de Liapunov
              \(V(x,y)\) cerca de un equilibrio satisface típicamente \(V\ge 0\) y
              \(\dot V=\nabla V\cdot (f,g)\le 0\).
              A veces se encuentran pre‑integrales (integrales primeras) \(H(x,y)\) con
              \(\frac{d}{dt}H(x(t),y(t))=0\). Entonces \(V=H^2\) es no‑creciente (de hecho constante).
            


            
Heurísticas del applet

            
    
              
  • Si \(\frac{dy}{dx}=\frac{g}{f}\) depende sólo de \(x\), entonces \(H=y-\int q(x)\,dx\) es constante.
    • 
              
  • Si \(\partial_y f=\partial_x g\), intenta reconstruir un potencial \(V\) con \((f,g)=-\nabla V\).

Idea

            

              For an autonomous system \(\dot x=f(x,y),\ \dot y=g(x,y)\), a Lyapunov function
              \(V(x,y)\) near an equilibrium typically satisfies \(V\ge 0\) and
              \(\dot V=\nabla V\cdot (f,g)\le 0\).
              Sometimes one finds first integrals \(H(x,y)\) with
              \(\frac{d}{dt}H(x(t),y(t))=0\). Then \(V=H^2\) is non‑increasing (in fact constant).
            


            
Applet heuristics

            
    
              
  • If \(\frac{dy}{dx}=\frac{g}{f}\) depends only on \(x\), then \(H=y-\int q(x)\,dx\) is conserved.
    • 
              
  • If \(\partial_y f=\partial_x g\), it tries to reconstruct a potential \(V\) such that \((f,g)=-\nabla V\).
