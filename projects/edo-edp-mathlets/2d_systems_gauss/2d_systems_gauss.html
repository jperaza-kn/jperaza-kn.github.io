Decomposición de Gauss


            

              Este applet visualiza el sistema lineal
              \[
                \dot{X}=AX,\qquad A=FDE,
              \]
              con
              \[
              F=\begin{pmatrix}1&f\\0&1\end{pmatrix},\quad
              D=\begin{pmatrix}d_1&0\\0&d_2\end{pmatrix},\quad
              E=\begin{pmatrix}1&0\\e&1\end{pmatrix}.
              \]
            


            


            
Cómo usar

            
    
              
  • Ajusta \(f,e,d_1,d_2\).
    • 
              
  • Haz clic en el gráfico para crear una trayectoria desde \((x_0,y_0)\).
    • 
              
  • ✖ elimina trayectorias.
    • 
            


            

              Este tipo de descomposición es fundamental en teoría de grupos y geometría diferencial.
            

          


          
          

            
Gauss decomposition


            

              This applet visualizes the linear system
              \[
                \dot{X}=AX,\qquad A=FDE,
              \]
              where
              \[
              F=\begin{pmatrix}1&f\\0&1\end{pmatrix},\quad
              D=\begin{pmatrix}d_1&0\\0&d_2\end{pmatrix},\quad
              E=\begin{pmatrix}1&0\\e&1\end{pmatrix}.
              \]
            


            


            
How to use

            
    
              
  • Adjust \(f,e,d_1,d_2\).
    • 
              
  • Click the plot to add a trajectory starting at \((x_0,y_0)\).
    • 
              
  • Use ✖ to remove trajectories.
    • 
            


            

              This decomposition is central in group theory and differential geometry.
